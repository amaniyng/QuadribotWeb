R-box M,
as a dashboard and instrument cluster

The brain of tomorrow's vehicles. Rbox model M is a small and compact, modular computer designed for urban mobility to allow for easy vehicle sharing system and as navigation.

Sensors for autonomous driving​

Cameras

The low-cost cameras on AHD can be connected via additional module that convert the AHD signal into serial CSI.

LIDAR​

Lidars, essential for autonomous driving can be connected via the ethernet port on the nVidia module or to the iMX port.

RADAR​

For long range or corner radars, connected via CAN-FD they are an essential element for cruise control and braking behind the front vehicles.

Ultra-sound

From autonomous parking or “summon” functionality, the use of ultra-sound sensors is less expensive than other solutions.​

Highlights​

Processors & rear modules​

iMX 6

For the industrial version of the RBox, modelM can accommodate an iMX6 low power as a module. This can be used in SCADA application or in low-power application where there is a medium processing demand.​

iMX 8

For the more demanding applications, iMX8 can be used for route calculation, video recording as a DVR and communicate with the external world. With the optional nVidia module it can be used in autonomous vehicles or for image processing.

iMX 8QX + DSP

For the most advanced version of the RBox-model M it can accommodate an iMX8QX with DSP for real-time processing. Along with the optional nVidia Xavier module, it can power autonomous platforms or allow for more powerful artificial intelligence algorithms to operate.​

nVidia Xavier NX

With a special placement inside the RBox, the nVidia Xavier NX can be used to add extra power for Artificial intelligence application or whenever there is a high-demand in processing power.​
